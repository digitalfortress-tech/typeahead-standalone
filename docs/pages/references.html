1. ‚ú® API

There are 3 methods exposed by typeahead-standalone

  1. reset(clearLocalSource?: boolean);

    Resets the typeahead instance to the state it was in before any user interaction. It removes all items from the search index except those that were added via a local source. To remove absolutely all items, the function accepts an optional parameter which should be set to true. Reset() also clears cached remote requests.

    JS 
        /* Setup typeahead */
    const instance = typeahead({ /* config options */ });

    /* clear search index except items added via a local source */
    instance.reset();

    /* clears entire search index */
    instance.reset(true);

    This API is useful in situations where you need to invalidate data after a certain time has elasped.

  2. addToIndex();

    Adds items to the search index. Intended to use after calling the reset() method. Similar to adding items via a Local source.

    JS 
            /* Setup typeahead */
        const instance = typeahead({ /* config options */ });

        instance.reset(true); /* or instance.reset(); */
        instance.addToIndex(['Blue, Baige , Black']);

    Can also be used to update the search index at a predestined time or after an interval.

  3. destroy();

    Destroys the typeahead instance, clears the search index, removes all event handlers and cleans up the DOM.

    JS 
        /* Setup typeahead */
    const instance = typeahead({ /* config options */ });

    /* Destroy typeahead */
    instance.destroy();

    Useful if you wish to deactivate typeahead completely.

i. How to execute something on clicking a suggestion ?
There are 2 ways to accomplish this -
JS 
  /**** 1. By adding an input event listener ****/

  // get the input element on which you have or will initialise typeahead
  const input = document.querySelector(".searchInput");
  input.addEventListener("input", (event) => {
    if (!event.isTrusted) { // check if event was triggered manually
      // execute something
      // alert("event type is - " + event.data);
    }
  });



  /**** 2. By using the display callback ****/
  typeahead({
    input: document.querySelector(".searchInput"),
    source: {
      // ...
    },
    display: (item, event) => {
      if (event) {
        // execute something
        // alert("event type is - " + event.type);
      }

      return `${item.name}, ${item.country}`;
    }
  });
