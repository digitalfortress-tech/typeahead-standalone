Install

There are several ways in which you can install the library. They are listed below...

1. Package Manager ðŸ“¦

NPM Yarn
Shell 
npm install --save typeahead-standalone
Shell 
yarn add typeahead-standalone

2. CDN

JSDelivr Unpkg

Include the Javascript

HTML 
<script src="https://cdn.jsdelivr.net/npm/typeahead-standalone"></script>

Include the CSS (Optional, we recommend to override the default styling)

HTML 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/typeahead-standalone/dist/basic.css" />

The above links, by default, point to the latest version of typeahead-standalone.js

Include the Javascript

HTML 
<script src="https://unpkg.com/typeahead-standalone@5.4.0/dist/typeahead-standalone.umd.js"></script>

Include the CSS (Optional, we recommend to override the default styling)

HTML 
<link rel="stylesheet" href="https://unpkg.com/typeahead-standalone@5.4.0/dist/basic.css" />

The above links can be used to link to a specific version of typeahead-standalone.js

The library is available at window.typeahead

3. Direct Download ðŸ“¥

To download the library, you can use the NPM registry. 

https://registry.npmjs.org/typeahead-standalone/-/typeahead-standalone-5.4.0.tgz
where 5.4.0 refers to the version number. To use another version, replace the version in the above url by the desired version. All available versions/releases are listed on github. The required JS and CSS is located in the dist/ folder.
