Playground

Listed below are several practical examples of typeahead-standalone.js. All code snippets are live, meaning that you are free to make any changes which will be reflected in the result tab.

1. Colors Example

This example lists almost every available config option. It uses a local data source. Templates are used to stylize the suggestions.

See the Pen Typeahead-standalone Colors Example by Niket Pathak (@niketpathak) on CodePen.

2. Movies Example

This example shows the usage of a prefetch data source. Here is the raw dataset at your disposal. It has been retrieved via the TMBD API.

See the Pen Typeahead-standalone Movies Example by Niket Pathak (@niketpathak) on CodePen.

If you notice, movies are searchable by their title. To be able to also search by their description, add the overview property to the keys config option above. On doing so, when you search for Barry Allen or Bruce Wayne, you should see the expected suggestions.

3. Github Users Example

This example shows the usage of a remote data source. Templates are used to stylize the suggestions.

See the Pen Typeahead-standalone Github users Example by Niket Pathak (@niketpathak) on CodePen.

Feel free to test the performance of this library with this demo page which has about 50k records to search from.
